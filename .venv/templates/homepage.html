51st Engineer Brigade, Philippine Army

Physical Fitness Test

Here are some components of the physical fitness test:

1. Push-up Test

A test used to measure upper body strength and endurance.

2. Sit-up Test

A test used to measure abdominal strength and endurance.

3. 1.5-Mile Run

A test used to measure cardiovascular endurance.

4. Flexibility Test

A test used to measure the range of motion of joints.
